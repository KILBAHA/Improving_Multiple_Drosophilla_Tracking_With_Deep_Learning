Directory used to generate the additional images for the enriched dataset. 

Coords.npz -> contains coordinates of flies generated by training

test.mp4 -> video used to test original model - needed to generate crops

enrich.py -> script used to create crops from test.mp4 added to enrich

enrich/curate_enriched.py -> script used to curate enriched fly images

enrich/enriched.info -> output from curate_enriched.py. Concatenated with description.info.curated to make combined.info in Haar/Training/Enrichment
